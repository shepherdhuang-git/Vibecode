# Vibecode

This repository contains a minimal static frontend prototype for a travel site called Vibecode.

What I added:
- `index.html` — homepage with hero, search bar, and sample destination cards
- `styles.css` — simple stylesheet for a clean, minimal design

Deploying on GitHub Pages:
1. Go to your repository Settings → Pages.
2. Under "Source", select the `main` branch and `/ (root)` folder, then Save.
3. After a minute or two, your site will be available at `https://shepherdhuang-git.github.io/Vibecode/` (if your GitHub username and repo name match).

Next steps you can request:

- Integrate a map (Leaflet) and lazy-loading for images.
